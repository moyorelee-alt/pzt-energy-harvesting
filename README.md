Project: Centrifugal Piezoelectric Harvester (Sub-QZS)
Author: Jianxiao Zheng
MATLAB Version used in creating codes: R2022a
1. What it does
Simulates a ball on a parabolic track under centrifugal instability, impacting
piezoelectric beams to harvest energy. Includes validation and optimization.
2. How to run
Open MATLAB R2024. Navigate to this folder. Type the script name and press Enter.
3. Run first
>> validation_script
4. Main files
- validation_script.m   : Baseline physics validation
- final_power_plot.m    : Speed sweep (10-70 rad/s)
- Gss_Optimization.m    : Parameter optimization
- get_parameters.m      : System parameters
